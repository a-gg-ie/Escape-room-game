# Escape Room Game

**Escape Room Game** is a Python-based puzzle game where players solve logical and visual puzzles to escape from different themed rooms. This game has two rooms:

- **Room 1: Artist's Room** – Solve painting and color-based puzzles to unlock the cabinet and escape.
- **Room 2: Temple** – Decode symbols and follow patterns to open the stone door and escape.

---

## Game Features

- Text-based interaction using Python terminal input.
- Multiple puzzles with logical and pattern-based challenges.
- Hints available for players who get stuck.
- Step-by-step progression leading to an ultimate escape.

---

## Gameplay Instructions

1. Run the Python script:
   ```bash
   python escape_room.py
2. Choose a room:
   Enter 1 for Room 1 (Artist’s Room)
   Enter 2 for Room 2 (Temple)
   Enter 0 to exit the game
3. Follow the on-screen instructions to solve puzzles:
   Input numeric or alphabetic codes.
Hints are provided if needed.
Final password is required to escape each room.
Room 1: Artist's Room
Solve the painting puzzle (sum what “shouldn’t be counted”).
Mix primary colors to get a number equal to the new color’s letter count.
Enter the final 3-digit password to escape.
Room 2: Temple
Decode symbols based on the scroll’s instructions.
Solve the tile pattern puzzle (Left → Middle → Right → Middle).
Enter the final alphanumeric password (following a Letter-Number pattern) to escape.
How to Play
Input codes as prompted.
Press Enter to continue in between instructions.
If an invalid input is entered, the game will ask again.
Use hints by entering 0 when prompted for certain puzzles.
